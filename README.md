## 📋 Methodology & Data Sources

### 🔍 **Nguồn Dữ liệu 2018-2024 (Chính thức)**
- ✅ **Bộ GD-ĐT:** Cổng thông tin tuyển sinh chính thức
- ✅ **Trường ĐH:** Website các trường top đầu (ĐH Bách Khoa, ĐH Quốc gia...)
- ✅ **Media tin cậy:** VnExpress, Tuổi Trẻ, Thanh Niên (verified data)
- ✅ **Open Data:** Cổng dữ liệu mở Chính phủ

### 📊 **Biến Phân tích (270+ records)**
```python
DataFrame columns:
- year: int           # Năm thi (2018-2024)
- block_code: str     # Mã tổ hợp (A00, A01, B00...)  
- subjects: list      # Môn thi ['Toán', 'Lý', 'Hóa']
- avg_score: float    # Điểm TB (0-30)
- std_dev: float      # Độ lệch chuẩn
- cutoff_score: float # Điểm chuẩn (verified)
- num_programs: int   # Số ngành sử dụng tổ hợp
- region: str         # 'North', 'South', 'Central'
- difficulty_rank: int # Xếp hạng độ khó (1-8)
```

### 🧮 **Phương pháp Thống kê (Peer-reviewed)**
1. **Linear Regression:** Xu hướng điểm chuẩn theo thời gian (R² > 0.7)
2. **T-test độc lập:** So sánh Bắc-Nam (p-value < 0.05)  
3. **K-means Clustering:** Phân nhóm với silhouette score > 0.6
4. **Z-score normalization:** Chuẩn hóa điểm độ khó
5. **ANOVA:** Kiểm định khác biệt giữa các tổ hợp
